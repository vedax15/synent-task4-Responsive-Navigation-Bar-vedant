# Responsive Navigation Bar 🧭

**Task 4**

A clean, fully responsive navigation bar built with HTML, CSS, and JavaScript. This project demonstrates mobile-first design principles, flexbox layouts, and interactive UI elements for a seamless user experience across all devices.

## 🚀 Features

* **Responsive Design:** Adapts smoothly to desktop, tablet, and mobile screen sizes.
* **Mobile Hamburger Menu:** Clean and intuitive toggle menu designed specifically for smaller screens.
* **Brand & Navigation:** Well-structured layout featuring a dedicated logo space and interactive menu items.
* **Smooth Toggling:** Utilizes JavaScript to seamlessly open and close the mobile drop-down menu.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure for the navigation bar and list elements.
* **CSS3:** Media queries for responsiveness, flexbox for alignment, and smooth hover/transition effects.
* **JavaScript (Vanilla):** Event listeners and DOM manipulation to handle the mobile menu toggle state.

## 💻 Getting Started

To view this project locally, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone <your-repository-url>
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd <your-repository-directory>
    ```
3.  **Run the project:**
    Simply open the `index.html` file in your preferred web browser. No local server or build tools are required!
